FAQ
Preguntas frecuentes
¿Necesito conocimientos técnicos para usar AnticiPalta? +
No. El sistema está diseñado para agricultores y técnicos agrícolas. Solo ingresa los datos de tu fundo y el sistema hace el resto.
¿Qué provincias cubre el sistema? +
Actualmente cubre Chincha (Ica) y Yauyos — Catahuasi (Lima), las principales zonas productoras de palta Hass del Perú.
¿Qué tan preciso es el modelo? +
El modelo XGBoost alcanza un R² de 0.9276, lo que significa que explica el 92.76% de la variación en el rendimiento real. El MAE promedio es de 20.89 toneladas.
¿Los datos climáticos son automáticos? +
Sí. Al seleccionar tu provincia y mes, el sistema precarga automáticamente los promedios históricos de temperatura, precipitación, humedad y radiación solar de SENAMHI. Puedes ajustarlos manualmente.
¿Puedo registrar múltiples fundos? +
Sí. Puedes registrar, editar y gestionar todos tus fundos desde la plataforma y generar predicciones independientes para cada uno.
